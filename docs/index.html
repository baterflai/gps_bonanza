CoolX4 Parameter Reference

B

BAT_CRIT_THR
Type: 4-bit Integer | Default: 5 | Min: 0 | Max: 15

Critical battery threshold as percentage of capacity. Triggers emergency landing when reached.

Bit Description
0-3 Threshold percentage (0-15, maps to 0-30%)
BAT_LOW_THR
Type: 4-bit Integer | Default: 10 | Min: 0 | Max: 15

Low battery warning threshold as percentage of capacity.

Bit Description
0-3 Threshold percentage (0-15, maps to 0-45%)

C

COM_ARM_EKF
Type: 4-bit Integer | Default: 1 | Min: 0 | Max: 15

Maximum EKF innovation threshold for arming. Higher values allow arming with less accurate state estimates.

Bit Description
0-3 Innovation threshold multiplier (0-15)
COM_FLTMODE
Type: 4-bit Integer | Default: 2 | Min: 0 | Max: 15

Default flight mode on boot.

Bit Description
0-3 0=Manual, 1=Altitude, 2=Position, 3=Mission, 4=Loiter
COM_RC_LOSS
Type: 4-bit Integer | Default: 3 | Min: 0 | Max: 15

Action on RC signal loss.

Bit Description
0-3 0=Disabled, 1=Loiter, 2=RTL, 3=Land, 4=Terminate

F

FILTER_FUSE_SRC
Type: 4-bit Integer | Default: 6 | Min: 0 | Max: 15

Bitmask to determine which sensors are fused into the state estimator.

Bit Description
3 Reserved
2 Fuse IMU1
1 Fuse IMU2
0 Fuse GPS

G

GEO_FENCE_ACT
Type: 4-bit Integer | Default: 1 | Min: 0 | Max: 15

Action when geofence is breached.

Bit Description
0-3 0=None, 1=Warning, 2=Loiter, 3=RTL, 4=Land
GEO_MAX_ALT
Type: 4-bit Integer | Default: 8 | Min: 0 | Max: 15

Maximum allowed altitude above home position.

Bit Description
0-3 Altitude limit (value × 50m, 0-750m)
GPS_AVAIL
Type: 4-bit Integer | Default: 1 | Min: 0 | Max: 15

Exposes the availability of the GPS hardware to the flight stack.

Bit Description
3 Reserved
2 Reserved
1 Reserved
0 GPS Hardware Available (1=Yes, 0=No)
GPS_PUB_FREQ
Type: 4-bit Integer | Default: 1 | Min: 0 | Max: 15

Sets the frequency (in Hz) at which the GPS module samples and publishes position data.

Bit Description
0-3 Frequency Value in Hz (0-15)

I

IMU_DGYRO_CUT
Type: 4-bit Integer | Default: 8 | Min: 0 | Max: 15

Gyroscope low-pass filter cutoff frequency.

Bit Description
0-3 Cutoff frequency (value × 10 Hz)
IMU_INTEG_RATE
Type: 4-bit Integer | Default: 4 | Min: 0 | Max: 15

IMU integration rate divider for state estimation.

Bit Description
0-3 Rate divider (IMU rate / value)

L

LND_FLIGHT_T
Type: 4-bit Integer | Default: 0 | Min: 0 | Max: 15

Minimum flight time before landing detection is enabled.

Bit Description
0-3 Time in seconds (0-15)

M

MAG_COMP_TYP
Type: 4-bit Integer | Default: 0 | Min: 0 | Max: 15

Magnetometer compensation type for motor interference.

Bit Description
0-3 0=Disabled, 1=Throttle, 2=Current, 3=Both
MIN_GPS_SAT_VAL
Type: 4-bit Integer | Default: 0 | Min: 0 | Max: 15

Minimum number of satellites required to fuse GPS data.

If the number of satellites reported by the GPS module is below this value, the GPS measurement will be rejected by the filter.

Bit Description
0-3 Minimum Satellite Count (0-15)
MOT_SLEW_MAX
Type: 4-bit Integer | Default: 0 | Min: 0 | Max: 15

Maximum motor slew rate limit.

Bit Description
0-3 0=Disabled, 1-15=Rate limit (%/10ms)
MPC_THR_HOVER
Type: 4-bit Integer | Default: 8 | Min: 0 | Max: 15

Hover throttle estimate used for altitude control.

Bit Description
0-3 Throttle percentage (value × 5%, 0-75%)
MPC_XY_VEL_MAX
Type: 4-bit Integer | Default: 12 | Min: 0 | Max: 15

Maximum horizontal velocity in position control mode.

Bit Description
0-3 Velocity limit (value m/s)
MPC_Z_VEL_MAX
Type: 4-bit Integer | Default: 5 | Min: 0 | Max: 15

Maximum vertical descent velocity.

Bit Description
0-3 Velocity limit (value × 0.5 m/s)

N

NAV_ACC_RAD
Type: 4-bit Integer | Default: 5 | Min: 0 | Max: 15

Waypoint acceptance radius for mission mode.

Bit Description
0-3 Radius in meters (0-15m)
NAV_RCL_ACT
Type: 4-bit Integer | Default: 2 | Min: 0 | Max: 15

Action when RC link is lost during mission.

Bit Description
0-3 0=Continue, 1=Loiter, 2=RTL, 3=Land

R

RC_CHAN_CNT
Type: 4-bit Integer | Default: 8 | Min: 0 | Max: 15

Number of RC channels to process.

Bit Description
0-3 Channel count (0-15)
RTL_RETURN_ALT
Type: 4-bit Integer | Default: 10 | Min: 0 | Max: 15

Return-to-launch altitude above home position.

Bit Description
0-3 Altitude (value × 10m, 0-150m)

S

SYS_AUTOSTART
Type: 4-bit Integer | Default: 4 | Min: 0 | Max: 15

Airframe configuration preset.

Bit Description
0-3 Airframe ID (see airframe reference)
SYS_MC_EST_GRP
Type: 4-bit Integer | Default: 2 | Min: 0 | Max: 15

Estimator group selection for multicopter.

Bit Description
0-3 0=Disabled, 1=LPE, 2=EKF2, 3=Q Estimator

T

TEL_FRSKY_CFG
Type: 4-bit Integer | Default: 0 | Min: 0 | Max: 15

FrSky telemetry configuration.

Bit Description
0-3 0=Disabled, 1-15=UART port number
THR_MDL_FAC
Type: 4-bit Integer | Default: 5 | Min: 0 | Max: 15

Thrust to motor mapping factor for non-linear ESCs.

Bit Description
0-3 Factor (value × 0.1, 0.0-1.5)
